# MNL Calculator

**Расчёт профиля давления и расхода жидкости для трубопровода с распыляющими форсунками.**

Веб-приложение предназначено для инженерного расчёта гидравлического профиля вдоль линии с дозирующими распыляющими форсунками. Модель учитывает потери давления по Дарси–Вейсбаху, режим течения (ламинарный, переходный, турбулентный) и нелинейную характеристику форсунок вида `Q = k · Pⁿ`.

## ✨ Возможности

- Расчёт давления на каждой форсунке от источника до заглушки.
- Расчёт расхода через каждую форсунку и суммарного расхода линии.
- Автоматический подбор давления у заглушки методом бисекции.
- Визуализация режимов течения на графике (ламинарное / переходное / турбулентное).
- Пороговая линия минимального давления корректного распыления.
- Автоматическая маркировка форсунок с критически низким давлением:
  - красные точки на графике,
  - красные строки в таблице результатов.
- Экспорт результатов в CSV и Markdown.
- Полностью клиентское приложение — работает без сервера (достаточно открыть `index.html`).

## 🔬 Математическая модель

### Потери давления в трубе

Потери давления на участке рассчитываются по уравнению Дарси–Вейсбаха:

```
ΔP = f · (ρ / 2) · (L / D) · v²
```

gде:
- `f` — коэффициент гидравлического трения,
- `ρ` — плотность жидкости,
- `L` — длина участка,
- `D` — внутренний диаметр трубы,
- `v` — скорость потока.

### Коэффициент трения

| Режим течения | Формула | Диапазон Re |
|---|---|---|
| Ламинарный | `f = 64 / Re` | Re < 2300 |
| Переходный | Линейная интерполяция | 2300 ≤ Re < 4000 |
| Турбулентный (гладкие трубы) | `f = C · 0.316 / Re^0.25` | Re ≥ 4000 |

`C` — эмпирическая **коррекция для турбулентного режима** (по умолчанию `1.0`). Если эксперимент показывает меньшие потери, уменьшите `C` (например, `0.90` или `0.85`).

### Характеристика форсунки

Расход через одну форсунку описывается степенной зависимостью:

```
Q = k · Pⁿ
```

- `n = 1.0` — типично для мелкодисперсных **вихревых форсунок** с тангенциальными входами (swirl atomizers), используемых в данном проекте.
- `n = 0.5` — классическое инерционное истечение через простой орбифис.

Коэффициент `k` автоматически пересчитывается из заданного расхода форсунки при давлении источника.

### Схема расчёта

Расчёт выполняется в два этапа:

1. **Внутренний проход** — от последней форсунки (у заглушки) к первой. На каждом шаге накапливается суммарный расход, и давление пересчитывается с учётом потерь на участке.
2. **Внешняя бисекция** — подбор давления у заглушки `P_after` таким образом, чтобы давление на входе в систему совпало с заданным давлением источника.

## 📊 Визуализация результатов

### Блоки результатов

После расчёта отображаются четыре ключевых показателя:

1. **Общее количество форсунок, шт**
2. **Суммарный расход через форсунки, л/мин**
3. **Давление на первой форсунке, бар**
4. **Давление в конце линии, бар**

### График

- Синяя линия — профиль давления вдоль трубопровода.
- Цветные фоны — режимы течения (зелёный / жёлтый / красный).
- Красная пунктирная линия — минимальное давление корректного распыления.
- **Красные точки** — форсунки, работающие ниже минимального давления распыления.

### Таблица

Таблица содержит данные по каждой форсунке:

- № форсунки
- Отметка трубопровода, м
- Давление на форсунке, бар
- Расход через форсунку, мл/мин
- Расход сегмента, л/мин

Строки с форсунками ниже порога распыления выделены красным цветом.

## 🔧 Калибровка по эксперименту

Если расчётные значения не совпадают с экспериментом, настройте два параметра:

1. **Показатель степени n форсунки** — определяет чувствительность расхода форсунки к давлению.
   - Для вихревых форсунок: `n ≈ 1.0`
   - Для простых дозаторов: `n ≈ 0.5`

2. **Коррекция для турбулентного режима** — подгоняет потери давления в трубе.
   - `1.0` — классический Блазиус.
   - Уменьшайте, если экспериментальное падение давления меньше расчётного (например, `0.90`, `0.85`).
   - Увеличивайте, если экспериментальное падение больше расчётного.
